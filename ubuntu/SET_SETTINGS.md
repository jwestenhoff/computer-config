## set shortcut for opening nautilus window
**Ubuntu Version**:

``` Ubuntu 18.04.4 LTS ```

**Where**:

``` System Settings -> Keyboard -> Shortcuts -> Custom shortcut ```

**command**:

```bash nautilus -w ```

**short cut**:

``` Super(Windows) + E ```

# Legend
- Nautilus =  file system explorer of ubuntu
- Super = Windows Key
